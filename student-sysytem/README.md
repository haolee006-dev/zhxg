# 智慧学工系统

## 项目介绍

智慧学工系统是一个基于 Vue + Spring Boot + MySQL 的全栈学生工作管理系统，包含学员信息管理、迎新管理、学生工作、宿舍管理、离校管理和数据分析等六大核心模块。

## 技术栈

- **前端**: Vue 2.x + Element UI + ECharts
- **后端**: Spring Boot 2.7.x + MyBatis Plus + JWT
- **数据库**: MySQL 8.0
- **JDK版本**: 21

## 快速开始

### 完整操作流程

#### 第一步：环境准备

1. **安装JDK 21**
   - 下载并安装JDK 21
   - 配置JAVA_HOME环境变量
   - 验证：`java -version`

2. **安装MySQL 8.0**
   - 下载并安装MySQL 8.0
   - 启动MySQL服务
   - 设置root密码（建议设置为root）

3. **安装Node.js 14+**
   - 下载并安装Node.js
   - 验证：`node -v` 和 `npm -v`

4. **安装Maven 3.6+**
   - 下载并安装Maven
   - 配置MAVEN_HOME环境变量
   - 验证：`mvn -version`

#### 第二步：数据库初始化

1. 打开MySQL客户端（如Navicat、MySQL Workbench或命令行）
2. 执行数据库脚本：
   ```bash
   # 方式1：使用命令行
   mysql -u root -p < student-system-backend/src/main/resources/db/schema.sql
   
   # 方式2：在MySQL客户端中直接执行schema.sql文件内容
   ```
   
   脚本会自动：
   - 创建 `student_system` 数据库
   - 创建所有数据表
   - 插入测试数据

3. 确认数据库创建成功

#### 第三步：配置后端

1. 进入后端目录：
   ```bash
   cd student-system-backend
   ```

2. 修改配置文件（如需要）：
   - 文件位置：`src/main/resources/application.yml`
   - 修改数据库用户名和密码（默认为root/root）
   - 如果Redis未安装，可注释掉Redis配置

3. 安装依赖并启动：
   ```bash
   # 清理并安装依赖
   mvn clean install
   
   # 启动Spring Boot应用
   mvn spring-boot:run
   ```
   
4. 验证后端启动成功：
   - 访问：http://localhost:8080/api/doc.html (Swagger文档)
   - 看到API文档页面即表示成功

#### 第四步：配置前端

1. 打开新的终端窗口，进入前端目录：
   ```bash
   cd student-system-frontend
   ```

2. 安装依赖：
   ```bash
   npm install
   ```
   
3. 启动开发服务器：
   ```bash
   npm run serve
   ```
   
4. 验证前端启动成功：
   - 访问：http://localhost:3000
   - 看到登录页面即表示成功

#### 第五步：登录系统

使用以下账号登录：

| 角色 | 用户名 | 密码 | 说明 |
|------|--------|------|------|
| 管理员 | admin | 123456 | 拥有所有权限 |
| 教师 | teacher | 123456 | 教师/辅导员权限 |
| 学生 | student | 123456 | 学生权限 |

### 各模块详细操作指南

#### 1. 学员信息管理模块

**学生档案管理：**
1. 登录系统，进入"学员信息管理" → "学生档案"
2. 点击"新增"按钮，填写学生信息：
   - 学号（必填）
   - 姓名（必填）
   - 性别、出生日期、身份证号
   - 学院、专业、班级、年级
   - 联系方式等
3. 点击"保存"完成添加
4. 在列表中可进行编辑、删除、查询操作

**辅导员管理：**
1. 进入"学员信息管理" → "辅导员管理"
2. 点击"新增"添加辅导员信息
3. 填写工号、姓名、学院、联系方式等
4. 保存后可管理辅导员信息

**违纪处分查询：**
1. 进入"学员信息管理" → "违纪处分"
2. 可按学生姓名、处分类型查询
3. 查看处分详情和状态

#### 2. 迎新管理模块

**预报到登记：**
1. 进入"迎新管理" → "预报到登记"
2. 点击"新增"进行预报到
3. 填写：
   - 学号、姓名
   - 预计到达日期
   - 交通方式（火车、飞机、汽车等）
   - 到达站点和时间
   - 行李数量
4. 提交后等待确认

**报到流程管理：**
1. 进入"迎新管理" → "报到流程"
2. 查看各项报到手续进度
3. 点击"更新"标记流程完成状态
4. 包含：入学资格审核、缴费、领取物资、入住宿舍等流程

#### 3. 学生工作模块

**学生证办理：**
1. 进入"学生工作" → "学生证管理"
2. 点击"新增"办理证件
3. 选择证件类型，填写证件号码
4. 设置有效期和状态

**奖学金管理：**
1. 进入"学生工作" → "奖学金管理"
2. 点击"新增"添加奖学金记录
3. 填写：
   - 奖学金名称（国家奖学金、校级奖学金等）
   - 类型和金额
   - 获奖学期和日期
4. 保存后可查看和管理

**资助管理：**
1. 进入"学生工作" → "资助管理"
2. 可管理：
   - 困难生认定
   - 助学金申请
   - 勤工助学岗位
   - 国家助学贷款
3. 提交申请后等待审批

**学情测评：**
1. 进入"学生工作" → "学情测评"
2. 填写德育、智育、体育分数
3. 系统自动计算总分和等级

**健康档案：**
1. 进入"学生工作" → "健康档案"
2. 记录学生体检信息
3. 包括身高、体重、视力、血压等

#### 4. 宿舍管理模块

**宿舍信息管理：**
1. 进入"宿舍管理" → "宿舍信息"
2. 查看楼栋、房间、入住情况
3. 可新增、编辑宿舍信息

**卫生检查：**
1. 进入"宿舍管理" → "卫生检查"
2. 点击"新增"记录检查结果
3. 选择宿舍、填写检查日期、评分
4. 记录检查结果和备注

**宿舍报修：**
1. 进入"宿舍管理" → "宿舍报修"
2. 学生提交报修申请：
   - 选择楼栋和房间
   - 选择报修类型（水电、家具、门窗等）
   - 详细描述问题
3. 管理员查看并处理
4. 更新状态为"处理中"或"已完成"

**宿舍违纪：**
1. 进入"宿舍管理" → "宿舍违纪"
2. 记录违纪情况：
   - 违纪类型（晚归、使用违章电器等）
   - 违纪描述和日期
   - 处罚措施
3. 处理后更新状态

**假期留校：**
1. 进入"宿舍管理" → "假期留校"
2. 学生提交留校申请
3. 填写假期名称、起止日期、留校原因
4. 管理员审批

#### 5. 离校管理模块

**离校办理：**
1. 进入"离校管理" → "离校办理"
2. 学生提交离校申请
3. 填写离校类型、日期、原因、目的地
4. 各级部门依次审批
5. 完成所有离校手续

**准毕业生管理：**
1. 进入"离校管理" → "毕业生管理"
2. 管理毕业生信息
3. 跟踪论文状态、就业状态
4. 记录毕业日期和证书编号

#### 6. 数据分析模块

**数据看板：**
1. 进入"数据分析" → "数据看板"
2. 查看各类统计图表：
   - 学生人数统计（按学院、年级）
   - 性别比例
   - 宿舍入住率
   - 奖学金发放统计
3. 支持图表切换和数据筛选

**学生画像：**
1. 进入"数据分析" → "学生画像"
2. 输入学号查询学生详情
3. 查看综合信息：
   - 基本信息
   - 成绩趋势
   - 获奖记录
   - 资助情况
   - 宿舍信息

## 常见问题

### 1. 后端启动失败
- 检查MySQL是否启动
- 检查数据库连接配置是否正确
- 检查JDK版本是否为21
- 查看控制台错误日志

### 2. 前端启动失败
- 检查Node.js版本是否符合要求
- 删除node_modules后重新npm install
- 检查端口3000是否被占用

### 3. 登录失败
- 确认数据库已正确初始化
- 检查用户名密码是否正确
- 查看后端日志确认认证服务正常

### 4. Redis连接失败
- 如果未安装Redis，可在application.yml中注释Redis配置
- 或安装并启动Redis服务

## 配置说明

### 后端配置 (application.yml)

```yaml
# 数据库配置
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/student_system?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: root

# JWT配置
jwt:
  secret: student-system-secret-key-2024
  expiration: 86400000
```

### 前端配置 (vue.config.js)

```javascript
devServer: {
  port: 3000,
  proxy: {
    '/api': {
      target: 'http://localhost:8080',
      changeOrigin: true
    }
  }
}
```

## 开发规范

### 后端规范
- 统一返回结果封装: Result<T>
- 统一分页结果封装: PageResult<T>
- 使用 MyBatis Plus 进行数据库操作
- 使用 JWT 进行身份认证
- 使用注解进行参数校验

### 前端规范
- 使用 Element UI 组件库
- 统一请求封装: request.js
- 使用 Vuex 进行状态管理
- 使用 Vue Router 进行路由管理
